# Gold vs Dollar Index Analysis (2015–2025)

This project analyzes the relationship between Gold Prices and the U.S. Dollar Index (DXY) using historical financial data and Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Statsmodels
- Google Colab

## Dataset
The dataset includes historical price data for:
- Gold Futures (GC=F)
- U.S. Dollar Index (DXY)

## Analysis Steps
1. Data Collection
2. Data Cleaning
3. Correlation Analysis
4. Data Visualization
5. Regression Analysis

## Key Result
The correlation between Gold and the U.S. Dollar Index during 2015–2025 is approximately **0.34**, indicating a weak positive relationship.
